# Magic Analysis · 命理全析

> 综合八字命理 · 西方星座 · 塔罗原型 · 道佛哲学，为你和伴侣生成客观、深度的命运分析报告。

**[→ 查看介绍页](https://latte-analyzer.github.io/magic-analysis/)**

---

## 这是什么

一个 [Claude Code](https://claude.ai/code) Skill（`/magic-analysis`）。

输入出生信息或八字排盘截图，自动生成两个文件：
- **`.md`** — 完整文字报告（5000–8000字）
- **`.html`** — 暗色 PPT 风格幻灯片（9页，含交互导航）

## 分析框架

| 传统 | 内容 |
|------|------|
| 八字命理 | 旺衰法·格局·喜忌神·大运流年·神煞 |
| 西方占星 | 太阳星座·外行星过运·合盘相位 |
| 塔罗原型 | 大阿卡纳人格牌·流年牌·关系牌 |
| 道佛哲学 | 阴阳五行·业力视角·具体化解方案 |

覆盖六大维度：格局/事业/财运/感情/健康/人生功课

## 安装

1. 确保已安装 [Claude Code](https://claude.ai/code)
2. 将 `SKILL.md` 放入你的 skills 目录：

```bash
mkdir -p ~/.claude/skills/magic-analysis
cp SKILL.md ~/.claude/skills/magic-analysis/SKILL.md
```

3. 在 Claude Code 对话中输入 `/magic-analysis` 启动

## 使用

```
/magic-analysis
```

然后提供：
- 出生日期 + 时间 + 性别
- 或直接粘贴八字排盘 App 截图（支持自动识别）

两个人的信息可以一起提供，Claude 自动分辨。

## 原则

- 每个维度强制正负平衡，不做选择性展示
- 每条风险结论引用具体命理术语，可自行验证
- 危机分 A/B/C 三级，给出可执行的化解步骤
