分析原则

客观，是这个分析的底线

命理分析容易变成"算命摊子"——只说好话，模糊预测。Magic Analysis 的规则恰好相反。

⚖️
强制正负平衡
每个分析维度都必须同时列出优势和风险。如果某个领域只有正面结论，说明分析不完整——必须重新审视。
🔍
引用具体命理术语
每个风险结论都要标注是哪个配置导致的，例如「午午自刑」「劫财+羊刃」「伤官见官」，让读者可以自行验证，而非盲信。
🎯
具体而非泛泛
不说"注意身体健康"，说"心血管系统，高危季节为午未月（6–8月），建议每年这个时段做心电图"。每条建议都要具体到可执行。
A 级危机
高概率 · 高影响
需要立即制定行动计划。例：劫财+羊刃年的破财与血光风险，提供具体防范步骤。
B 级危机
中概率 · 中影响
提前预警，给出观察指标。例：职场是非年，需要减少公开批评性发言。
C 级危机
低概率 · 值得留意
提升意识即可，不需要过度焦虑。例：飞刃丧门的离别关，知道就好，做好心理准备。
使用方式

三步，拿到你的报告

在 Claude Code 里，一句话启动，其余交给 Claude。

一
输入命令
在 Claude Code 对话框输入 /magic-analysis，Claude 会自动加载分析框架，进入引导流程。
二
提供信息
提供出生日期、时间、性别——或直接把八字排盘 App 的截图丢进去（支持 h5.40dhjien.cn 等主流排盘 App 格式），Claude 会自动识别所有字段。两个人的信息都可以一起提供，Claude 会自动分辨谁是谁。
三
获取双文件
分析完成后，.md 报告和 .html 幻灯片会自动保存到你指定的文件夹（默认：图片所在目录）。直接打开 HTML 文件即可浏览完整的交互式报告。
